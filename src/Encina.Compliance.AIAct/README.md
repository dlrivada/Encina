# Encina.Compliance.AIAct

Declarative EU AI Act (EU 2024/1689) compliance at the CQRS pipeline level. Risk classification, prohibited practices detection, human oversight enforcement, and transparency obligations — enforced consistently across HTTP, messaging, gRPC, and serverless transports.

## Quick Start

```csharp
// 1. Register services
services.AddEncinaAIAct(options =>
{
    options.EnforcementMode = AIActEnforcementMode.Block;
    options.AutoRegisterFromAttributes = true;
    options.AssembliesToScan = [typeof(Program).Assembly];
});

// 2. Decorate your request types
[HighRiskAI(Category = AISystemCategory.EmploymentWorkersManagement, SystemId = "cv-screener")]
[RequireHumanOversight(Reason = "Employment decisions require human review")]
[AITransparency("This screening result was generated by AI.")]
public sealed record ScreenCandidateCommand(string CandidateId) : ICommand<ScreeningResult>;

// 3. The pipeline behavior enforces compliance automatically
// - Validates risk classification
// - Checks for prohibited practices (always blocked)
// - Evaluates transparency obligations
// - Returns EncinaError if non-compliant (Block mode) or logs warning (Warn mode)
```

## Features

| Feature | EU AI Act Article | Description |
|---------|-------------------|-------------|
| **Risk Classification** | Art. 6, Annex III | 4 risk tiers with 12 high-risk categories |
| **Prohibited Practices** | Art. 5 | 8 prohibited practices, always blocked regardless of enforcement mode |
| **Human Oversight** | Art. 14 | Decision recording, approval tracking, review requirements |
| **Transparency** | Art. 13, 50 | 5 obligation types (AI content, deepfakes, emotion, biometric, chatbot) |
| **Technical Documentation** | Art. 11 | Design specs, data governance, accuracy metrics |
| **Data Quality** | Art. 10 | Training data validation, bias detection, gap analysis |
| **Auto-Registration** | — | Scan assemblies for `[HighRiskAI]` at startup |

## Attributes

### `[HighRiskAI]`

Marks a request type as a high-risk AI operation (Art. 6, Annex III).

```csharp
[HighRiskAI(Category = AISystemCategory.CriticalInfrastructure)]
public sealed record MonitorPowerGridCommand(string GridId) : ICommand<GridStatus>;

// With explicit system ID (overrides auto-generated ID)
[HighRiskAI(Category = AISystemCategory.BiometricIdentification, SystemId = "face-match-v2")]
public sealed record IdentifyPersonCommand(byte[] FaceData) : ICommand<IdentityMatch>;
```

### `[RequireHumanOversight]`

Requires human review before the request is executed (Art. 14).

```csharp
[RequireHumanOversight(Reason = "Loan decisions above €50k require human approval")]
public sealed record ApproveLoanCommand(string ApplicationId, decimal Amount) : ICommand<LoanDecision>;
```

### `[AITransparency]`

Declares transparency obligations (Art. 13/50).

```csharp
[AITransparency("This content was generated by AI.", ObligationType = TransparencyObligationType.AIGeneratedContent)]
public sealed record GenerateReportCommand(string Topic) : ICommand<Report>;
```

## Configuration

```csharp
services.AddEncinaAIAct(options =>
{
    // Enforcement mode: Block (reject), Warn (log), Disabled (no-op)
    options.EnforcementMode = AIActEnforcementMode.Block;

    // Auto-register AI systems from [HighRiskAI] attributes at startup
    options.AutoRegisterFromAttributes = true;
    options.AssembliesToScan = [typeof(Program).Assembly];

    // Health check (opt-in)
    options.AddHealthCheck = true;
});
```

## Enforcement Modes

| Mode | Behavior | Prohibited Practices |
|------|----------|---------------------|
| **Block** | Non-compliant requests return `EncinaError` | Always blocked |
| **Warn** | Non-compliant requests proceed with log warning | Always blocked |
| **Disabled** | No compliance checks (pipeline passthrough) | Always blocked |

> **Important**: Prohibited practices (Art. 5) are **always blocked** regardless of enforcement mode. Social scoring, real-time biometric surveillance, and other prohibited uses cannot be bypassed.

## Error Codes

| Code | Description |
|------|-------------|
| `aiact.system_not_registered` | AI system not found in registry |
| `aiact.prohibited_use` | Prohibited practice detected (Art. 5) |
| `aiact.compliance_validation_failed` | Compliance violations found |
| `aiact.human_oversight_required` | Human review required but not provided |
| `aiact.transparency_required` | Transparency obligation not met |
| `aiact.classification_failed` | Risk classification failed |
| `aiact.registry_lookup_failed` | Registry lookup error |
